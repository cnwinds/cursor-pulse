# 临时 Key 借用

额度不足时，成员可在私聊自助借用团队富余 Cursor Key。

## 常用说法

- `借 Key` / `借 key`
- `归还 Key`
- `我的借用`

## 执行提示

- 借用：tool `key_loan_request`
- 查看本人借用：tool `key_loan_self_read`（按 `tasks/my-loan.md` 展示版式排版）
- 归还：tool `key_loan_return`
