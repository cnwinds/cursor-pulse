# 绑定钉钉工作群

将机器人拉入目标群后，在群内 **@ 机器人** 并发送：

```
启动
```

同义：`激活`、`开始`、`开工`、`start`

**作用：**
- 将当前群设为 **工作群**（月报、催办等群消息发到此群）
- 配置写入管理后台「钉钉集成」

**权限：**
- 首次绑定：群内任意成员可发送「启动」
- 更换工作群：仅管理员

**说明：** 此操作由 Channel 本地处理，无对应 Agent tool。
